# Analise de Dados de Redes Sociais com Base em Consultas de Grafos

## Problema

Uma Startup de análise de mídia sociais deseja criar um novo produto que ofereça insights sobre o engajamento e as conexões entre usuários em uma plataforma.

Eles precisam de um protótico funcional que possa responder a perguntas conplexas sobre interações de usuários, popularidade de conteúdo e comunidades de interesse.
