# Desafio

Você foi contratado para desenvolver um sistema de recomendação de músicas que utiliza grafos para identificar padrões de esculta e sugerir novas faixas aos usuários. O sistema deve ser capaz de:
- Representar usuários, músicas, artistas e gêneros como nós em um grafo.
- Representar inteações (como escutar, curtir, seguir) como arestas com propriedades.
- Utilizar consultas Cypher para gerar recomendações personalizadas com base em conexões no grafo.

## Técnicas possíveis para conexões no grafico
- Queries
- GDS > PageRank
- Similaridade
- Comunidades
- Caminhos
